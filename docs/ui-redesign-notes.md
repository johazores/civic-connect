# UI Redesign Notes

## Design Direction

The design system uses a premium civic SaaS direction with:

- White and off-white surfaces
- Blue primary actions
- Teal/emerald trust accents
- Slate typography
- Soft borders and shadows
- Mobile-first layouts
- Consistent card and form patterns

## Navigation

Public navigation uses a desktop menu and a mobile hamburger overlay. Staff navigation uses clear top-level tabs for Reports, Content, and Settings.

## CRUD Pattern

The staff content workspace follows a consistent pattern:

1. Choose content type.
2. Create or edit in the form area.
3. Review records in the library.
4. Edit or archive from each record card.

This keeps service, hotline, news, category, department, and staff user management predictable.
