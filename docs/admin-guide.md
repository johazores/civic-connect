# Admin Guide

This guide explains the staff portal workflow.

## Sign In

Open:

```text
/metro-city/admin/login
```

Use staff credentials provided by the organization administrator.

## Reports Workspace

The Reports tab is the main operations workspace.

Staff can:

- Search reports by reference, citizen name, issue, or location
- Filter by status, category, department, and priority
- Select a report from the queue
- Review citizen details, location, description, and uploaded photos
- Assign a department
- Update status
- Set priority
- Add public updates for citizens
- Add internal notes for staff
- Export filtered reports to CSV

## Content Workspace

The Content tab has one consistent CRUD pattern:

1. Choose a content type.
2. Fill out the form on the left.
3. Save the record.
4. Review saved records in the library on the right.
5. Edit or archive records from the library.

Supported content types:

- Services
- Hotlines
- News
- Report categories
- Departments
- Staff users

## Settings Workspace

The Settings tab controls tenant identity and public contact details:

- Organization name
- City/province name
- Tagline
- Description
- Email
- Phone
- Brand color
- Address

## Operational Notes

- Public updates are visible in citizen tracking.
- Internal notes remain in the staff timeline.
- Departments and categories are tenant-scoped.
- Archived content is hidden from the public portal.
- Replace starter passwords before real use.
