# User Guide

This guide explains how citizens use the public portal.

## Open the Portal

Use a tenant URL such as:

```text
/metro-city
/laguna-province
```

The public navigation includes Home, Report, Track, Services, Hotlines, News, and Account.

## Create an Account

1. Open `/metro-city/register`.
2. Enter full name, email, phone, and password.
3. Submit the form.
4. The portal redirects to the citizen dashboard.

A citizen account is useful because submitted reports are saved automatically in the dashboard.

## Sign In

1. Open `/metro-city/login`.
2. Enter the citizen email and password.
3. The dashboard shows report history, active reports, resolved reports, and direct tracker links.

## Report a Concern

1. Open `/metro-city/report`.
2. Choose the report category.
3. Enter contact details.
4. Add a clear title and description.
5. Add the location.
6. Upload a photo when helpful.
7. Submit the report.
8. Save the generated reference number.

The request can be tracked immediately after submission.

## Track a Request

1. Open `/metro-city/track`.
2. Enter a reference number such as `MCS-2026-0001`.
3. Review status, department assignment, public updates, and attachments.

Status flow:

```text
Submitted → Reviewing → Assigned → In Progress → Resolved
```

## View Services

Open `/metro-city/services` to browse available public services and responsible offices.

## View Hotlines

Open `/metro-city/hotlines` to see emergency and service contacts. On mobile, tap the call button to dial.

## View News

Open `/metro-city/news` for public announcements. Each news card opens a detail page.
